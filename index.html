Elastica Documentation

The goal of this pages is to add some documentation for the Elastica client.

To edit the documentation to have to do the following: 

$ git clone git://github.com/ruflin/Elastica
$ cd Elastica
$ git fetch origin
$ git checkout -b gh-pages origin/gh-pages
Then you can start creating your changes.

API

The API documentation of the current Elastica master can be found here. The API is automatically created from the source code

You can download this project in either zip or tar formats.
